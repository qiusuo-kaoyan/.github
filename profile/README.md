> 求索号是一艘渡所有考研学子上岸的船。

---

## 我们是谁？

「求索号考研舰队」是一个**代际传承的开源考研知识社群**。第一代舰长 Leo 将个人 Obsidian 考研笔记库开源，后续每一代上岸的学弟学妹都可以 Fork、优化、拓展，并通过 Pull Request 将改进反哺母舰。

**核心特色不是笔记本身，而是「人机协同考研」的方法论：**
- 通过自研的 **PZ 系列 Obsidian 模板**（P = Prompt，Z = Zenith）与 AI 协同工作
- 将错题思路、课程要点输入模板，由 AI 辅助结构化输出
- 再经人工审校后汇入母舰，形成可迭代的知识循环
